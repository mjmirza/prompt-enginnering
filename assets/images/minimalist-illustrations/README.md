# Minimalist Conceptual Illustrations

This directory contains reference images for the minimalist conceptual illustration style with infographic elements.

## About This Style

This visual style is characterized by:
- Hand-drawn sketch aesthetic
- Minimalist composition with clean white/light backgrounds
- Conceptual metaphors representing abstract ideas
- Infographic elements (arrows, labels, diagrams)
- Limited color palettes (earthy tones, muted colors)
- Human-touched, non-futuristic feel

## Reference Documentation

For prompt templates and detailed guidelines on creating images in this style, see:
- `/03-patterns/minimalist-conceptual-illustrations.md`

## Image Categories

Organize reference images by template type:

### core-concept/
Images demonstrating core concepts through central metaphors (e.g., weighing scales for balance, keys for unlocking potential).

### process-transformation/
Images showing processes, changes, or transformations (e.g., tangled to organized, complex to simple).

### challenge-barrier/
Images illustrating obstacles, challenges, or barriers (e.g., protected gems, constrained ideas).

### human-interaction/
Images emphasizing human control, guidance, or interaction (e.g., hands connecting elements).

### text-visual-metaphor/
Images combining text with visual metaphors to reinforce specific messages.

## Usage

These reference images serve multiple purposes:
- Style inspiration for creating new visuals
- Examples when explaining the aesthetic to others
- Reference for consistency across generated content
- Templates for similar conceptual illustrations

## Adding References

When adding new reference images:
1. Place in the appropriate category subdirectory
2. Use descriptive names: `template-type-description-##.png`
3. Note the source or generation method
4. If generated with AI, save the prompt used in a companion `.txt` file
5. Add brief descriptions or annotations if helpful

## File Naming Convention

Format: `[category]-[description]-[number].[extension]`

Examples:
- `core-concept-weighing-scale-01.png`
- `process-transformation-knot-to-thread-01.png`
- `human-interaction-hands-connecting-02.png`
